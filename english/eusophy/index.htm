Basic Concepts in Logic 1

Intellection is the activity of knowing the objective world, and intellect (human intelligence) is the capacity to know that world. Accordingly, the objective world is the totality of intellectual objects. An intellectual object is called an entity if it is meaningful to ask whether the object exists. For instance, ''UFOs'' and ''Peter'' (in ''The Tail of Peter Rabbit'') are entities. An intellectual object is called an event if it is meaningful to ask whether the object occurs. For instance, ''UFOs are round'' and ''Peter eats a radish'' are events, as well as their parts ''are round,'' ''eats a radish,'' and ''eats.'' Not all but many intellectual objects may be expressed by language. Linguistic expressions referring to entities are called nominals, while those referring to events are called declaratives. Observation of language and introspection show that the intellectual objects are divided into entities and events. Observation of language also shows that each event is either an attribute of an entity (and the attribute itself), or a relation between entities (and the relation itself). For instance, the event ''UFOs are round'' is an attribute of the entity ''UFOs'' and the event ''are round'' is the attribute itself. The event ''Peter eats a radish'' is a relation between the entities ''Peter'' and ''radish,'' and the event ''eats'' is the relation itself. The event ''Peter eats a radish'' is also an attribute of the entity ''Peter'' and the event ''eats a radish'' is the attribute itself, which is also an attribute of the entity ''radish,'' showing that someone eats ''radish,'' and the event ''eats'' is also the attribute itself. Therefore, each event A may involve some entities a1, ..., an for some positive integer n, so that A is sometimes denoted by A(a1, ..., an). Here a1, ..., an may not be all of the entities involved. For instance, if ''Peter eats a radish,'' ''Peter,'' and ''radish'' are denoted by A, b, and c, respectively, then A may also be denoted by A(b), A(c), A(b,c), and A(c,b).

Let A1, ..., An, and B denote events, where n is a nonnegative integer; if n = 0, A1, ..., An do not exist. Then the following five expressions are equivalent by definition:

  1. If all of A1, ..., An occur, then B occurs.
  2. The tuple (A1, ..., An) implies B; that is, A1, ..., An together imply B.
  3. A1, ..., An together are connected to B by causality (causation, causal relation, cause-effect relation).
  4. B is an effect of the causes A1, ..., An.
  5. A1, ..., An ⇒ B.

This definition in particular says that the symbolic expression A1, ..., An ⇒ B in 5 should be read as in 1 through 4, especially as ''A1, ..., An together imply B.'' By definition, the expressions 1 through 5 are also equivalent to the following one, provided n > 1:

  1. A1 implies B under A2, ..., An.

Because of these definitions, the expressions 1 through 6 are also equivalent to the expressions obtained by arbitrary permutation of A1, ..., An. To understand causality in Hypothesis 1 below means finding out the reason why A1, ..., An ⇒ B; that is, why A1, ..., An together imply B, for some tuple (A1, ..., An, B) of events. When n = 0 in the expression A1, ..., An ⇒ B, it is denoted by () ⇒ B, and B is said to be inevitable or, in religious context, B is said to have been determined by Providence (God's will). For instance, the event ''x is x'' is inevitable for an arbitrary entity x. Obviously, if B is inevitable, then A1, ..., An ⇒ B for every tuple (A1, ..., An) of events. If B is not inevitable, we say that B is evitable or avoidable. For instance, the event ''x is not x'' is avoidable for an arbitrary entity x. A law is an event which is supposed to be inevitable. Causality is nontrivial; that is, the following holds:

  1. There exists a tuple (A1, ..., An, B) of events which does satisfy A1, ..., An ⇒ B.
  2. There exists a tuple (A1, ..., An, B) of events which does not satisfy A1, ..., An ⇒ B.
Thus, our universe is a cosmos governed by a profound causality.

Basic Concepts in Logic 2

Observation of language shows that there exist certain operations on the events. Each operation assigns an event to each tuple (A1, ..., An) of events, for a positive integer n called the arity of the operation. If the operation is denoted by F, then the event which F assigns to A1, ..., An is denoted by F(A1, ..., An). For each event A, -A denotes the negation of A. The symbol ''−'' stands for the expression ''It is not the case that'' and therefore is prepositive (incidentally, it should be postpositive in the Japanese word order). If A denotes, say, the event ''UFOs are round,'' then -A is the event ''It is not the case that UFOs are round,'' which is equivalent to the event ''UFOs are not round.'' For each pair (A, B) of events, A & B denotes the conjunction of A and B. For instance, if A and B denote the events ''Peter eats a radish'' and ''Peter eats a lettuce,'' respectively, then A & B is the event ''Peter eats a radish and Peter eats a lettuce,'' which is equivalent to the event ''Peter eats a radish and a lettuce.'' The negation operation assigns each event A its negation -A. It is a unary operation and denoted by the symbol ''-,'' hence -(A) = -A. The conjunction operation assigns each pair (A, B) of events their conjunction A & B. It is a binary operation and denoted by the symbol &, hence &(A,B) = A & B. The negation operation and the conjunction operation are basic ones. There is a variety of operations, because operations can be composed. For instance, composing the negation operation and the conjunction operation, we obtain the operation which assigns each pair (A, B) of events the event -(-A & -B), although it is equivalent to the basic event ''A OR B.''

Let F be an operation of arity n. Then F is called a deduction if A1, ..., An ⇒ F(A1, ..., An); that is, if A1, ..., An together imply F(A1, ..., An), for each tuple (A1, ..., An) of events. For instance, the conjunction operation & is a deduction, while the negation operation ''-'' is not a deduction, because ''x is x'' does not imply -(''x is x''), or more generally, no inevitable event implies -(''x is x'').

Let F be an operation, n be its arity, and C be a collection of events. Then F(C) denotes the collection of all events F(A1, ..., An) for all tuples (A1, ..., An) of events in C. An event B is said to be generated by C or C-generated if the following holds:

For some positive integer n, there exists a chain (sequence) A1, ..., An of events such that An is equivalent to B, and for each integer k from 1 through n, the event Ak satisfies one of the following two conditions:

  1. Ak is a member of C (indeed, A1 is a member of C, because it does not satisfy the following condition).
  2. Ak is a member of Fk({A1, ..., Ak-1}) for some deduction Fk, where {A1, ..., Ak-1} denotes the collection of A1, ..., Ak-1.

Such a chain is called a C-chain for B. Obviously, if B is generated by C, then it is also generated by every collection larger than C. If Ak satisfies the condition 1, then there exists an event Ck in C such that Ak = Ck. If Ak satisfies the condition 2, then there exists a tuple (A1', ..., Ak') such that Ak = Fk(A1', ..., Ak') for some numbers 1', ..., k' less than k. The totality of Ck, Fk, and (A1', ..., Ak') for all integers k from 1 through n is called the C-explanation for B. Thus, if a C-chain for B exists, then a C-explanation for B exists, and vice versa. Thus, the following holds:

Proposition on Generation and Explanation:
Let C be a collection of events, and B be an event. Then B is C-generated if and only if there exists a C-explanation for B.

Let L be a collection of laws. If L has the following property called omnipotence, then we say that L is omnipotent:

The following two expressions are equivalent for each tuple (A1, ..., An, B) of events:

  1. B is an effect of the causes A1, ..., An.
  2. B is generated by the union L ∪ {A1, ..., An}.

If L has the following property called omniscience, then we say that L is omniscient:

The following two expressions are equivalent for each tuple (A1, ..., An, B) of events:

  1. A1, ..., An together imply B.
  2. There is an (L ∪ {A1, ..., An})-explanation for B.

Then ''Proposition on Generation and Explanation'' shows that the following proposition holds:

Proposition on Omnipotence and Omniscience:
Omnipotence and omniscience on a collection of events are equivalent; that is, it is omnipotent if and only if it is omniscient.

If a collection of events has these equivalent properties, we say that it rules (governs) causality.

§3. Noological Set Theory

A clarification of my view on sets is in order here, prompted by the following statement in Answer to Question 2:

In the mathematical science mentioned above, the nooworlds are defined as sets, whereas the noocosmos cannot be defined as a set due to a certain set-theoretical reason.

The set theory presented here is neither naïve nor axiomatic, but rather noological.

The noological definition of sets:
A set is defined as a collection of entities within a perceived range. These entities are gathered by our intellect, and the set itself is regarded as an entity.

This definition is noological because of the italicized words ''intellect,'' ''perceived,'' and ''entities,'' — an intellectual object is called an entity if it is meaningful to ask whether it exists.

Now, we can proceed as in naïve set theory. However, there exist certain differences. Notably, the following holds:

Proposition on belonginess:
Every set is distinct from its elements; that is, it does not belong to itself.
This is because each set is constructed by our intellect, which cannot include the set itself among the entities it gathers in the act of construction. This is analogous to the fact that a dish cannot include itself among its own ingredients prior to being made.

Entities other than sets are called prime entities. For example, the symbol I is a prime entity, as are all of its sequences such as I, II, III, and so on. We call the sequences I, II, III, ... the natural numbers as the ancient romans probably did, and as usual, denote them respectively by 1, 2, 3, ... in the decimal system. Using them, we can construct the sets of integers, rational numbers, real numbers, and so on, in a well-known way. The numbers are all prime entities. Thus, our set theory is open in that it takes prime entities into consideration.

Proposition on entitic conditions (conditions on entities):
If an entitic condition C satisfies the following condition 1, then it must also satisfy the following condition 2:
  1. There exists the set of all entities satisfying C.
  2. For some set X, every set containing X does not satisfy C.
However, the converse is not always true; that is, there exists an entitic condition C which satisfies the condition 2 but does not satisfy the condition 1.
Proof of the first assertion.
Suppose that an entitic condition C satisfies the condition 1. Then there exists the set A of all entities satisfying C. By way of contradiction, assume that C does not satisfy the condition 2. Then there exists a set B which contains A and satisfies C. The set B is an entity satisfying C, and so is an element of A. Since A is contained in B, B is an element B, contrary to Proposition on belonginess.
Proof of the second assertion.
By way of contradiction, assume that every entitic condition satisfying the condition 2 also satisfies the condition 1. Let p be a prime entity. Define the entitic condition Cp so that an entity x satisfies it if and only if x = p. Then every set does not satisfy Cp, and so Cp satisfies the condition 2. Therefore, Cp satisfies the condition 1 by the above assumption; that is, there exists the set {p} consisting only of p. Define the entitic condition C so that an entity x satisfies it if and only if x is a set whose elements are not prime entities. Then every set containing {p} does not satisfy C, and so C satisfies the condition 2. Therefore, C satisfies the condition 1 by the above assumption; that is, there exists the set A of all sets whose elements are not prime entities. But then, A is a set whose elements are not prime entities, and so A belong A, contrary to Proposition on belonginess.

An entitic condition is said to be unbounded if it does not satisfy the condition 2 in Proposition on entitic conditions. Namely, an entitic condition C is unbounded if, for each set X, there exists a set which contains X and satisfies C. Proposition on entitic conditions, then, shows that there does not exist the set of all entities satisfying C. This has various consequences about mathematics:

  1. There does not exist the set of all entities. This is because the empty entitic condition is obviously unbounded. Due to this, we cannot define noocosmos as a set, as was stated in Answer to Question 2
  2. There does not exist the set of all sets. To see this, define the entitic condition C so that an entity x satisfies it if and only if x is a set. Then C is obviously unbounded. Due to this and Proposition on belonginess, we can forget about the so-called Russel's paradox in naïve set theory. Also, we cannot develop the theory of cardinalities of sets as in naïve set theory.
  3. There does not exist the set of all well-ordered sets, and similarly for totally ordered sets and ordered sets. This is because every set is well-ordered by Zermelo's theorem.
  4. There does not exist the set of all groups, and similarly for monoids and semigroups. This is because, for an arbitrary given set X, there exists the free group whose basis is equal to X.
  5. For an arbitrarily given field K, there does not exist the set of all K-algebras, and similarly for K-vector spaces and rings. This is because every group G gives rise to K-group algebra whose basis is equal to G.

Since I mentioned Russel's paradox, some remarks on paradoxes may be in order.

Basic Concepts in Logic 3

An event A is said to be paradoxical if both A and its negation -A are inevitable; that is, if () ⇒ A and () ⇒ -A. In fact, no paradoxical event exists. This is verified by using the following logical laws:

  1. If A ⇒ B and B ⇒ C for events A, B, and C, then A ⇒ C.
  2. If -A ⇒ B for events A and B, then () ⇒ A OR B.
  3. If () ⇒ A OR -B for events A and B, then B ⇒ A.
  4. If () ⇒ A OR A for an event A, then () ⇒ A.
  5. -(-A) ⇒ A for each event A.

It is a good exercise to derive the following laws from the above ones:

  1. If -A ⇒ -B for events A and B, then B ⇒ A.
  2. If -A ⇒ A, then () ⇒ A.
  3. If A ⇒ -A, then () ⇒ -A.

The law 6 was used in the proof of Proposition on entitic conditions. The above-mentioned fact may be stated and proved as follows:

Proposition on paradoxes:
No paradoxical event exists.
Proof:
Suppose A is a paradoxical event. Then () ⇒ A and () ⇒ -A, and so X ⇒ A and -Y ⇒ -A for arbitrary events X and Y; the latter yields A ⇒ Y by the law 6, hence X ⇒ Y by the law 1. However, for instance, ''x is x'' ⇒ -(''x is x'') does not hold for every entity x.

This proof and certain logical laws show that there exists a paradoxical event if and only if X ⇒ Y for each pair (X,Y) of events.

I define a paradox as an argument which is incorrect because its conclusion is a nonexistent paradoxical event. Incidentally, ''para-'' in particular means ''incorrect'' and ''dox'' derives from ''dokein'' meaning ''to think.''

Examples of paradoxes

Example 1. Achilles does not catch up a tortoise:
Achilles run a race with a tortoise. It starts a meters ahead of him. He and it run b meters and c meters a second, respectively, and b is longer than c. Then the distance between him and it shorten b-c meters a second, and so becomes zero after a/(b-c) seconds; that is, he catches up it at that time. On the other hand, he arrives at its starting point after a/b seconds, when it is ac/b meters ahead of him. He arrives at the point after ac/b2 seconds, when it is ac2/b2 meters ahead of him. He arrives at the point after ac2/b3 seconds, when it is ac3/b3 meters ahead of him. ... Therefore, it is always ahead of him, and so he does not catch up it.

This asserts that both the event ''He catches up it'' and the event ''He does not catch up it'' are inevitable, hence ''He catches up it'' is paradoxical. There must be a mistake here by Proposition on paradoxes. There is no mistake before ''On the other hand.'' There is a mistake after that. The time in which it is ahead of him is less than (a/b + ac/b2 + ac2/b3 + ・・・) seconds, which is equal to a/(b-c) seconds; that is, the time in which he catches up it. Therefore, the description after ''On the other hand'' until ''...'' in fact means ''It is ahead of him until he catches up it,'' which is correct. The mistake exists in replacing ''until he catches up it'' with ''always.''

Example 2. There is a declarative which is both false and not false:
Consider the declarative ''This declarative is false.'' If this declarative is false, then it is not false. If it is not false, then it is false.

Let A denote the event ''This declarative is not false.'' Then this argues that -A ⇒ A and A ⇒ -A, hence it is paradoxical by the laws 7 and 8 above. There must be a mistake here by Proposition on paradoxes. Let B denote the first underlined expression ''this declarative.'' Then the second underlined expression ''this declarative'' refers to ''B is false.'' Let C denote it. Then the above argument in fact shows that B is false if and only if C is not false, hence B is not equal to C. The above argument, however, seems to assume that B is equal to C because the word ''this declarative'' is ambiguous.

Example 3. Inexpressible entities are expressible:
Let a denote one of the entities which are inexpressible. Then a is expressible as ''one of the entities which are inexpressible,'' and so a is not one of the entities which are inexpressible. Thus, a is not a.

This asserts that the event ''a is not a'' is inevitable, hence paradoxical, because the event ''a is a'' is also inevitable. There must be a mistake here by Proposition on paradoxes. In fact, after expressing an entity as ''inexpressible entity,'' it is lying as ''it is an inexpressible entity.'' There is another mistake. Every entity is expressible, even by one letter, as has been shown by the above sentence ''Let a denote one of the entities which are inexpressible.'' Similar remarks apply to the paradoxes obtained by replacing ''express'' by ''explain'' and ''define.''

If x is an entity and A is a set, the expression ''x ∈ A'' means ''x belongs to A.'' Russel's paradox in naïve set theory is as follows:

Let A be the set of all sets X which do not satisfy X ∈ X. Then, if A satisfies A ∈ A, then A does not satisfy A ∈ A by the definition of A. If A does not satisfy A ∈ A, then A satisfies A ∈ A by the definition of A.

Let B denote the event ''A does not satisfy A ∈ A.'' Then Russel asserts above that -B ⇒ B and B ⇒ -B, hence B is paradoxical by the laws 7 and 8 above. There must be a mistake here by Proposition on paradoxes. As shown above, it was found out in noological set theory. It is a pity that the initiators of naïve set theory could not find out it and proceeded to axiomatic set theory.

§4. Noocosmos, Noosystem, and Noolanguage

Just as we possess internal organs for digestion and circulation, we must also possess a neuro-system for intellection — this I call the noosystem. Considering that this system operates electrochemically, I begin with the following hypotheses:

Hypotheses on the noosystem and perception:
Hypothesis 1. The noosystem consists of certain electrochemical elements, which I refer to as perceptors.
Hypothesis 2. An element of a nooworld is perceived when a perceptor transforms into another kind of electrochemical element in response to that nooworld element. This transformed element is called a percept of the nooworld element, and the nooworld element is called the object of the percept.

I refer to this relation between nooworld elements and their percepts as the perception relation,

These hypotheses can be illustrated by analogy with a computer monitor. A monitor consists of numerous dots, which may be likened to perceptors. Consider the cover page of this site: apart from the title and the menu, it displays a picture of a landscape against a black background. Each dot in the picture corresponds to a unique point in the landscape — these are analogous to percepts. In contrast, the dots in the background correspond to nothing in particular and are analogous to perceptors that are not currently percepts.

''What are these electrochemical elements?'' I do not know the answer — nor, I suspect, does anyone. Nevertheless, we can attempt to study intellection by constructing and analyzing a mathematical model of it. This approach is, in spirit, akin to that of Gregor Mendel in the 19th century. Mendel constructed and analyzed a mathematical model of heredity. While his model involved certain ''elements,'' ours involves ''electrochemical elements.'' He observed pea plants to build his model; we observe language to build ours. Perhaps he hoped that someone, someday, would answer the question ''What are the elements?'' Likewise, we may hope that someone will one day answer the question ''What are the electrochemical elements?''

We refer to the expressions of percepts as noowords (not to be confused with nooworlds), and we refer to the totality of noowords as the noolanguage. Let A, B, and C respectively denote an element of a nooworld in the noocosmos, one of its percepts, and one of its expressions. Then we call B and A, respectively, a meaning and a semantic object of C; conversely call C a representation of A. These relations are illustrated by the following diagram:

Since each percept has a unique object, the number of the meanings of C is equal to that of the semantic objects of C.
